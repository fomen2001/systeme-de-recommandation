# 01 - Imports et bibliothèques

Objectif
- Lister et expliquer les bibliothèques nécessaires pour le notebook.

Contenu
- `zipfile` : gestion d'archives (non utilisé explicitement dans la suite, utile si les données sont zippées).
- `pandas as pd` : manipulation des tableaux et lecture des fichiers (`DataFrame`).
- `numpy as np` : fonctions numériques et génération aléatoire.
- `matplotlib.pyplot as plt` : visualisations (histogrammes, barplots).

Notes
- Ces imports sont standards pour l'exploration et le traitement des datasets en Python.
- Assurez-vous d'avoir installé `pandas`, `numpy` et `matplotlib` (par ex. `pip install pandas numpy matplotlib`).
